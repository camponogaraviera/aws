# Introduction

[Amazon Simple Email Service (SES)](https://aws.amazon.com/ses/) allows sending high-volume email notifications without relying on an on-premises Simple Mail Transfer Protocol (SMTP), instead, it uses the Amazon SES API or SMTP interface.
