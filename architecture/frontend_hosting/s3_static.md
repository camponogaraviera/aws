# Introduction

[Amazon S3](https://aws.amazon.com/s3/) is suitable for hosting **static websites** with no server-side scripting, only client-side. However, it requires the S3 bucket to be publicly readable (not recommended). It does not provide an SSL/TLS certificate for endpoints (HTTPS).

**Note:** Static websites mean HTML/CSS/JavaScript files with no backend logic and possibly APIs called from elsewhere. Example: landing page websites.
