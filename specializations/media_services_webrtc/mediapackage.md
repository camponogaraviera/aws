# Introduction

[AWS Elemental MediaPackage](https://aws.amazon.com/pt/mediapackage/) serves the video content into adaptive bitrate streaming formats (e.g., HLS, DASH, CMAF) and enables DVR features (pause, resume, start-over) using offset.
