# [Amazon SageMaker](https://aws.amazon.com/pm/sagemaker)

Sagemaker is a serverless service used to build, train, and deploy AI models.

SageMaker supports:

- Pipelines and Workflows: SageMaker allows you to orchestrate and automate ML workflows, from data preprocessing to model training and deployment.
- Model Training and [Fine-Tuning](https://docs.aws.amazon.com/sagemaker/latest/dg/jumpstart-fine-tune.html): train models on large datasets and fine-tune hyperparameters to improve performance.
- Real-time Inference: provides endpoints to deploy trained models for real-time inference, enabling low-latency predictions.
- [Batch Transform Jobs](https://docs.aws.amazon.com/sagemaker/latest/dg/batch-transform.html#:~:text=Batch%20transform%20automatically%20manages%20the,look%20like%20the%20following%20example.): for preprocessing and inference of large datasets without the need for a persistent endpoint.

# [Amazon Bedrock](https://aws.amazon.com/bedrock)

Bedrock is a serverless service that provides production-ready foundation models that can be fine-tuned with custom data to build generative AI applications.
