# About

[Amazon Neptune](https://aws.amazon.com/pt/neptune/) is a graph database. It is vendor lock-in and, therefore, integrates seamlessly with other AWS services. Available query languages are Apache TinkerPop Gremlin and SPARQL. Neptune is ACID-compliant.

# Pricing

Amazon Neptune has an on-demand instance pricing. You pay for:

1. Storage: billed per GB-month.
2. I/Os: billed per million requests.
3. Compute instance.
