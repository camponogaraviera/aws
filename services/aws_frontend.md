# [AWS Amplify](https://aws.amazon.com/amplify/)

AWS Amplify is a development platform that helps developers build secure, scalable mobile and web applications, including Progressive Web Apps (PWAs). It provides frontend libraries and UI components, as well as support for SSL/TLS certificates (HTTPS). 

Note: HTTPS relies on the TLS protocol (previously on SSL which is now deprecated) to establish encrypted communication between the client and the server.

Amplify is suitable for connecting the frontend of static websites (Angular, ReactJS, Next.js, or Vue.js/Nuxt.js) and mobile applications (kotlin, swift, flutter, react-native) with the backend.

Amplify abstracts much of the complexity of configuring and integrating AWS services such as AppSync (GraphQL APIs), Cognito (authentication), S3 (file storage), and Lambda (serverless functions).

Amplify has a no-code alternative ([Amplify Studio](https://aws.amazon.com/pt/amplify/studio/)) for building the frontend of apps by using a visual interface.
