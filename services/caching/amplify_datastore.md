# About

[AWS Amplify DataStore](https://aws.amazon.com/amplify/datastore/) is a local data store that enables on-device data storage and synchronization with support for offline capabilities. DataStore can cache and synchronize data retrieved from AWS services through AWS AppSync, including photo metadata or other app data, ensuring seamless offline functionality and automatic data syncing when online.
