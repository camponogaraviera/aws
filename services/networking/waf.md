# About

[AWS Web Application Firewall (WAF)](https://aws.amazon.com/pt/waf/) is used to protect the API from common exploits/attacks such as SQL injection (SQLi) and cross-site scripting (XSS). AWS CloudFront connects to AWS WAF.
