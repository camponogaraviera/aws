# [AWS AppSync](https://aws.amazon.com/appsync/)

A fully managed service that makes it easy to develop GraphQL APIs. Can handle the heavy lifting of securely connecting to various AWS services such as AWS DynamoDB, Lambda, and others.

# AWS Amplify

Used to host static websites and connect the frontend with the backend (AppSync, and other services).
